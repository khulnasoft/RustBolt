# 品牌指南

在这里你可以找到 Rustbolt 的品牌指南、资源和许可证。

## 名称

Rustbolt 在独立使用的情况下都应该写作 Rustbolt，而不是 rustbolt 或者 RUSTBOLT，但在命令行或者包名中可以使用小写，如 `@rustbolt/cli`。

## Logo

Rustbolt 的 Logo 是一个如闪电一样迅速的小螃蟹，代表了 Rustbolt 的快速编译。
